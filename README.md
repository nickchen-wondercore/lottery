# 尾牙抽獎系統

基於 Matter.js 物理引擎的互動式抽獎應用程式，模擬真實的球體容器抽獎機制。

## 功能特色

- 物理引擎驅動的球體滾動與碰撞效果
- 噴泉式雙渦流氣流系統（底部上吹，頂部左右分流）
- 風場粒子視覺化，即時呈現氣流方向
- 可調整球大小、字大小、亂流強度與抽籤間隔
- 出球自動選取離出口最近的球
- 支援多次連續抽籤
- 中獎名單即時顯示

## 技術棧

- **Matter.js 0.20.0** — 2D 物理引擎（CDN 載入）
- **Canvas 2D API** — 自訂繪製（非 Matter.js 內建 renderer）
- **純前端** — 無框架、無建置工具，靜態檔案直接運行

## 使用方式

1. 編輯 `names.json` 填入抽獎名單（JSON 字串陣列）
2. 用任意靜態伺服器啟動：
   ```bash
   python3 -m http.server 8080
   # 或
   npx -y http-server ./ -p 8080
   ```
3. 瀏覽器開啟 `http://localhost:8080/`

## 操作說明

| 控制項 | 說明 |
| ------ | ---- |
| **入籤筒** | 將名單中的球放入容器 |
| **數量** | 本次抽籤要抽出的球數 |
| **抽籤** | 開始抽獎 |
| **間隔** | 每顆球彈出的間隔秒數（1-10） |
| **氣流** | 噴泉式雙渦流強度（1-100，值越大越激烈） |
| **球大小** | 球體半徑（8-50），出口管寬度會自動適配 |
| **字大小** | 球上文字大小（0 = 依球大小自動計算，1-40 為手動指定） |
| **重置** | 清除結果，重新開始 |
