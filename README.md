# 尾牙抽獎系統

基於 Matter.js 物理引擎的互動式抽獎應用程式，模擬真實的球體容器抽獎機制。

## 功能特色

- 物理引擎驅動的球體滾動與碰撞效果
- 可調整亂流強度與抽籤間隔
- 支援多次連續抽籤
- 中獎名單即時顯示

## 技術棧

- **Matter.js 0.20.0** — 2D 物理引擎（CDN 載入）
- **Canvas 2D API** — 自訂繪製（非 Matter.js 內建 renderer）
- **純前端** — 無框架、無建置工具，靜態檔案直接運行

## 使用方式

1. 編輯 `names.json` 填入抽獎名單（JSON 字串陣列）
2. 用任意靜態伺服器啟動：
   ```bash
   npx serve .
   # 或
   python3 -m http.server 8080
   ```
3. 瀏覽器開啟 `http://localhost:8080/`

## 操作說明

| 控制項 | 說明 |
| ------ | ---- |
| **入籤筒** | 將名單中的球放入容器 |
| **數量** | 本次抽籤要抽出的球數 |
| **抽籤** | 開始抽獎 |
| **間隔** | 每顆球彈出的間隔秒數（1-10） |
| **氣流** | 亂流旋轉強度（1-10，值越大越激烈） |
| **重置** | 清除結果，重新開始 |
